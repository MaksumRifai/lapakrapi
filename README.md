# LapakRapi
Web-App LapakRapi ini digunakan oleh petugas keamanan untuk membantu dalam mendata pedagang
dalam membayar iuran, penerapannya hanya petugas yang dapat mengakses, untuk menginput pembayaran
mengecek status pembayaran melalui kode pedanang, maupun melihat laporan riwayat pembayaran.


Generate Gambar Code QR pedagang di
https://maksumrifai.github.io/lapakrapi-qr/

Input Kode Pedagang dan Nama Pedagang pada kolom

Contoh:
GOR001 Gorengan Aneka Rasa
POT001 Es Potong Rasa Memori

Setelah QR berhasil digenerate dapat kita Screenshoot atau Unduh.

Lalu gunakan Hasil Gambar QR tersebut di App LapakRapi
https://maksumrifai.github.io/lapakrapi/

Cukup scan atau unggah gambar QR tersebut untuk melakukan update status pembayaran
setelah petugas mendapatkan pembayaran dari pedagang saat penagihan iuran.
